# Blahg 2.0

## Overview

Blahg 2.0 is an open-source blog project built using Next.js. It provides a platform to share articles, thoughts, and ideas in a modern web environment.

## Features
- **Static Site Generation**: Fast performance and SEO friendly.
- **Dynamic Routing**: Create pages dynamically based on the content.
- **Markdown Support**: Write posts in Markdown for rich text formatting.
- **API Integration**: Easily integrate with external APIs to bring dynamic data.
- **User Authentication**: Secure login and registration functionality.
- **Responsive Design**: Works seamlessly on both desktop and mobile devices.

## Setup Instructions
1. **Clone the repository:**  
   ```bash
   git clone https://github.com/004Ongoro/blahg-2.0.git
   cd blahg-2.0
   ```  

2. **Install dependencies:**  
   ```bash
   npm install
   ```  

3. **Run the development server:**  
   ```bash
   npm run dev
   ```  
   Open your browser and navigate to `http://localhost:3000` to see your blog in action!
